# 06 — Combining PowerPoint & Canva: Prompts

Use these to plan your visual system and catch consistency problems before they compound.

---

## Setting up a visual system

```
I am building a quarterly review deck for corporate leadership. I want to use PowerPoint for the data and content slides and Canva for 3 visual slides (cover, key message, closing summary). Help me define a simple visual system I can apply consistently across both tools. I need decisions on: fonts, colours, text alignment, and image style.
```

```
My PowerPoint template uses these settings: [describe fonts, colours, slide dimensions]. I want to build 3 slides in Canva that will look visually consistent when inserted into this PowerPoint deck. What should I match exactly and what can I adapt?
```

---

## Catching consistency issues

```
I have finished combining my PowerPoint and Canva slides into one deck. Give me a checklist of things to review before I call it done — focusing on visual consistency between slides from the two tools.
```

```
Here are the descriptions of 5 consecutive slides in my deck: [describe each slide briefly — fonts, colours, layout style, image type]. Identify where visual inconsistency is most likely to be visible and what to fix.
```

---

## Managing the workflow

```
I need to update two slides in my deck that were originally designed in Canva and inserted as PNG images. The data has changed. Walk me through the most efficient workflow for updating these slides without redesigning them from scratch.
```

```
I have been working on a deck across two sessions and I have lost track of which slides are PowerPoint originals and which are Canva exports inserted as images. How can I tell the difference in PowerPoint, and does it matter for editing?
```

---

## Font and colour matching

```
My PowerPoint deck uses Calibri (headings: 24pt, body: 14pt) and these colours: navy #1E2761, teal #028090, white background, dark grey text #2D2D2D. I want to build a cover slide in Canva that matches this system. What font choices in Canva will come closest to Calibri, and how do I set custom colours in Canva free tier?
```

```
I used a Canva template that has a beautiful dark navy background. My PowerPoint slides all have white backgrounds. What are my options for making the deck look intentional rather than inconsistent when these slides appear side by side?
```
