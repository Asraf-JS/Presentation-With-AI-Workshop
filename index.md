# AI-Assisted Presentation Design Workshop

Notes, prompts, and exercises for the 2-day hands-on workshop.

---

## How to Use This Site

Each module has a **Notes** page covering the concepts and a **Prompts** page with copy-paste AI prompts you can use during and after the session.

To copy a prompt: open the prompts page, find the prompt you want, and click the copy icon on the top right of the code block.

To download everything: click the green **Code** button on the main repo page, then **Download ZIP**.

## Workshop Scenario

Throughout this workshop you work on one connected project from start to finish.

> **You are improving your department's quarterly review presentation for Nusantara Capital.**

Your department head has asked you to take last quarter's deck and turn it into something clearer, more visual, and more persuasive before it goes to senior leadership. Each module moves the project one step forward.

*If you are delivering this at your own organisation, treat "Nusantara Capital" as a stand-in for your company. The scenario works the same way.*

| Stage | What happens |
|-------|-------------|
| Rethink | Identify what is wrong with the existing deck and plan how to fix it |
| Structure | Reframe the message around a clear narrative and key takeaway |
| Rewrite | Use PowerPoint AI features to improve slide text and generate speaker notes |
| Visualise | Replace weak tables and charts with cleaner, story-driven visuals |
| Design | Bring in Canva to improve the visual quality of key slides |
| Integrate | Merge the Canva visuals back into the PowerPoint deck consistently |
| Review | Use AI to critique the full deck and identify what still needs work |
| Deliver | Present the improved deck and get peer feedback |
