# 05 — Canva for Visual Design: Prompts

Use these in **ChatGPT, Copilot Chat, or any AI chat tool** to plan and improve your Canva work.

---

## Deciding which slides to redesign in Canva

```
I have a 15-slide quarterly review deck for senior leadership. I want to redesign 2-3 slides in Canva to improve visual impact. Which types of slides typically benefit most from a Canva redesign versus staying in PowerPoint? Give me criteria I can use to decide.
```

```
Here are my 15 slide titles: [paste your titles]. Which 3 slides would most benefit from a Canva visual treatment, and why?
```

---

## Finding and adapting templates

```
I need a Canva template for a quarterly results summary slide. The slide needs to show: one large revenue number, a comparison to target, and 3 supporting bullet points. What should I search for in Canva's template library and what should I look for in a layout?
```

```
I found a Canva template I like but it has too many design elements — icons, decorative shapes, multiple colour blocks. I want to simplify it while keeping the layout structure. Give me a step-by-step approach for stripping back a busy Canva template without losing the grid and spacing.
```

---

## Matching brand colours

```
My organisation's primary colour is navy blue (#1E3A5F) and the accent colour is gold (#C9A84C). I am building a slide in Canva. Suggest a colour palette that works with these two colours for a professional corporate presentation — including what to use for backgrounds, body text, and highlights.
```

```
I am trying to match my company's PowerPoint template colours in Canva. The PowerPoint template uses these colours: [paste hex codes or describe colours]. How do I apply them manually in Canva free tier, and what do I need to watch out for when moving between the two tools?
```

---

## Exporting from Canva to PowerPoint

```
I have designed a cover slide in Canva and I want to insert it into my PowerPoint deck as the first slide. What is the best export format and resolution to use from Canva free tier so the image looks sharp on a projected screen?
```

```
I exported a Canva slide as PNG and inserted it into PowerPoint. It looks blurry when I project it. What are the likely causes and how do I fix it?
```

---

## Using Canva's AI features (free tier)

```
Canva has a Magic Design feature that generates designs from a text description. I want to create an infographic slide showing: the 4 stages of our project approval process. Write a text prompt I can use in Canva Magic Design to get a useful starting layout.
```

```
I used Canva's AI to generate a slide design but the output looks generic and does not match our brand. What are 3 ways I can use AI assistance in Canva more effectively to get results that are closer to what I need?
```
