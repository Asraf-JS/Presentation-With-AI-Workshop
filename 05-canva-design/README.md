# 05 — Enhancing Visual Design with Canva

Canva is not a replacement for PowerPoint. It is a tool that handles certain visual tasks better than PowerPoint does — and knowing which tool to use for which job saves a lot of frustration.

## When to Use Canva vs PowerPoint

The short answer: use PowerPoint for the structure and content of a full deck, use Canva for visual assets and individual slides where design quality matters.

| Task | Better in PowerPoint | Better in Canva |
|------|---------------------|-----------------|
| Full deck structure and flow | ✓ | |
| Animations and transitions | ✓ | |
| Presenter view and speaker notes | ✓ | |
| Complex data charts (linked to Excel) | ✓ | |
| Cover slides and title slides | | ✓ |
| Infographic slides | | ✓ |
| Visual summaries with icons | | ✓ |
| One-pager or visual callout slides | | ✓ |
| Consistent brand templates (free tier) | | ✓ |

For the Nusantara Capital deck, a practical split would be: build the structure in PowerPoint, then use Canva for the cover slide, one or two key message slides, and any infographic-style summary slides.

## Applying Canva Templates to Existing Content

Canva's free tier includes thousands of presentation templates. The challenge is adapting them to your content without just swapping in your text and hoping it works.

### A more useful approach

**Step 1 — Find a template for the type of slide, not the topic.**
Search for "business quarterly review" or "corporate results slide" rather than something too specific. You are looking for a layout that suits your content structure, not a theme that matches your industry.

**Step 2 — Strip back the template's default text.**
Most templates are over-designed with too many text boxes and decorative elements. Delete everything that does not serve your content. You are using the layout, not the template's original copy.

**Step 3 — Match your organisation's colours before anything else.**
Before changing any text, set the primary and secondary colours to match your brand. In Canva free, this is done manually by entering hex codes for each element. Get this right first — everything else will look more coherent.

**Step 4 — Adapt the layout to fit one clear point.**
Each Canva slide should still follow the same rule as a PowerPoint slide: one point, clearly stated in the title.

## Exporting and Integrating Canva Visuals into PowerPoint

Once you have built slides or visual assets in Canva, you need to bring them into your PowerPoint deck without quality loss.

### Recommended export approach

For **whole slides** you built in Canva and want to insert into PowerPoint:
- Export as PNG at 300 DPI (free tier: use standard export, then insert as an image in PowerPoint)
- Or export as PDF (single page) and crop on insertion

For **visual assets** (icons, infographic elements, illustration pieces):
- Export individual elements as PNG with transparent background (free tier has limited transparency options; use SVG export if available)
- Insert into your PowerPoint slide as an image layer on top of your existing content

### What to watch out for

**Font substitution.** Canva uses web fonts. When you export as PNG and insert into PowerPoint, the fonts are baked into the image — no substitution issue. But if you export as PPTX from Canva (paid feature), fonts may not carry over correctly on Windows.

**Resolution mismatch.** Standard Canva exports at 96 DPI by default. On a large projected screen this can look soft. Use the "Download — File type — PNG" option and check that the output dimensions match your slide size (typically 1920 × 1080px for widescreen).

**Colour shift.** Canva displays colours in sRGB for screen. If your organisation has specific Pantone or CMYK brand colours, the RGB equivalent in Canva may look slightly different from your PowerPoint template. Check both side by side before finalising.

## Free Tier vs Canva Pro

This workshop uses Canva's free tier. A few things to know:

| Feature | Free | Pro |
|---------|------|-----|
| Templates | Thousands (some locked) | All templates |
| Background remover | Not included | Included |
| Brand kit (saved colours/fonts) | Not included | Included |
| Transparent PNG export | Limited | Full |
| Magic Design AI | Included | Included |
| Resize to different formats | Not included | Included |

For a workshop or one-off project, the free tier is sufficient. For ongoing client work where brand consistency matters, Pro is worth considering.
