# 03 — Improving Slide Content with PowerPoint AI Features

PowerPoint has built-in AI features that can help you clean up slide text, improve layouts, and generate speaker notes. This module covers how to use them well, and when to ignore their suggestions.

> **Prompts to Try:** Open the [copy-paste prompt exercises](./prompts) for this module.

---

## Rewriting Slide Text for Clarity and Brevity

The most common slide problem is too many words. AI can help trim this, but you need to direct it, not just accept whatever it produces.

### The headline-first rule

Every slide title should state the point of that slide, not label the topic.

| Weak (label) | Strong (point) |
|--------------|----------------|
| Q3 Revenue | Revenue exceeded target for the second quarter running |
| Project Status Update | Two projects need a decision this week |
| Team Overview | Headcount is stable but capacity is not |
| Key Risks | Three risks require senior attention before Q4 begins |

Rewriting titles this way forces you to know what the slide is actually saying. If you cannot write an assertive title, the slide may not have a clear point yet.

### Using AI to trim slide content

When pasting slide content into an AI tool, be specific about what you want.

**Too vague:**
```
Improve this slide.
```

**Useful:**
```
This is a bullet list from a PowerPoint slide. Rewrite it so each bullet
is one short sentence, maximum 10 words. Remove anything that restates
what the title already says. The audience is senior leadership.
```

## Using PowerPoint Designer

PowerPoint Designer (Home tab, then Designer) suggests layout options when you add content to a slide. It works best for slides with a single image and short text, comparison slides with two or three items, and simple lists that benefit from visual structure.

**When to use it:** Designer is a starting point. Accept a suggestion when it genuinely improves the slide. Do not accept suggestions just because they look more polished — sometimes the simpler layout is clearer.

**When to ignore it:** Designer tends to add visual complexity to slides that should be simple. A slide with one big number does not need an elaborate layout. A slide making one strong point does not need decorative icons.

**Key limitation:** Designer cannot fix a slide with a bad message. It can rearrange a poorly structured slide into a more visually appealing poorly structured slide.

## Generating Speaker Notes with AI

Speaker notes are where the spoken content lives. Most people either leave them blank or paste their full script in. Neither is useful.

Good speaker notes do three things: state the point of the slide in one sentence, give 2 to 3 talking points the presenter should hit, and flag any transitions or things to watch out for.

### How to generate useful notes

Rather than asking AI to "write speaker notes for this slide", give it the context it needs:

```
Here is a slide from a quarterly review presentation. Title: "Revenue
exceeded target, Q3 performance at a glance". Bullet points: RM 4.2M
actual vs RM 3.8M target, 10% above plan, best Q3 in 4 years.

Write speaker notes for this slide. The presenter is a department head
speaking to the CEO and two division heads. Tone: confident but not
boastful. Time on this slide: about 90 seconds. Include a transition to
the next slide, which covers project risks.
```

The notes AI produces from a prompt like this are much more usable than what you get from a vague request.

## Practical Workflow for This Module

Work through the Nusantara Capital deck using this sequence:

1. Fix the title first. Rewrite each slide title to state a point, not a label.
2. Cut the bullets. Aim for no more than 3 bullet points per slide, each under 10 words.
3. Run Designer. Accept suggestions that improve clarity. Reject suggestions that add noise.
4. Write speaker notes. Use the AI prompt template above to generate notes for at least 3 key slides.

You do not need to do every slide perfectly. Focus on the 5 to 6 slides that carry the most weight in the deck.
